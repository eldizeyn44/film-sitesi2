# 🎬 Film Öneri Sitesi

Bu proje, **Web Programlama Ödevi** kapsamında geliştirilmiştir.  
Amaç: Dinamik PHP sayfaları kullanarak film öneri ve yorum paylaşım sitesi oluşturmak.

## 📂 Proje Yapısı
- `index.php` → Ana sayfa, menü ve en çok izlenen filmler
- `about.php` → Hakkında sayfası
- `films.php` → Film listesi
- `form.php` → Öneri yapma formu
- `comments.php` → Yorumları listeleme ve ekleme
- `save_comment.php` → Yorumları kaydetme
- `contact.php` → İletişim formu
- `data.txt` → Kullanıcı yorumlarının saklandığı dosya
- `style.css` → Tasarım dosyası

## 🛠️ Kullanılan Teknolojiler
- **PHP** (dinamik sayfalar, form işlemleri, dosya tabanlı veri saklama)
- **HTML & CSS** (tasarım ve arayüz)
- **XAMPP** (Apache + PHP çalıştırma ortamı)

## ✅ Ödev Gereksinimleri
- 5+ sayfa (index, about, films, form, comments, contact)
- Dinamik içerik (film listesi, yorumlar)
- Veri saklama (`data.txt`)
- Yönetim işlemi (yorum ekleme)

## 🚀 Çalıştırma
1. Proje dosyalarını `htdocs/zeynep` klasörüne kopyalayın.
2. XAMPP üzerinden Apache’yi başlatın.
3. Tarayıcıda `http://localhost/zeynep/index.php` adresini açın.

## 🎯 Sonuç
Bu site ile kullanıcılar:
- Popüler filmleri görebilir,
- Yeni film önerileri yapabilir,
- Yorumlarını paylaşabilir,
- İletişim formu ile mesaj gönderebilir.
